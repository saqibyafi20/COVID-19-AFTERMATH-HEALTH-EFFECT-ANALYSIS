# COVID-19-AFTERMATH-HEALTH-EFFECT-ANALYSIS
# Post-COVID Health Impact Analysis (SQL Project)

## Project Overview
This project analyzes post-COVID health outcomes of 500 patients to identify<br>
recovery patterns, hospitalization impact, symptom prevalence, and long COVID risk<br>
using SQL.

## Dataset
- 500 patient records<br>
- Features include age, gender, severity, hospitalization status,<br>
  recovery days, fatigue level, and long COVID risk.<br>

## Tools Used
- SQL (Data analysis & querying)<br>
- Power BI (optional â€“ visualization stage)<br>

## Key Business Questions
- What is the average recovery duration after COVID?<br>
- Does hospitalization increase recovery time?<br>
- Which patients experienced the longest recovery?<br>
- How does severity relate to long COVID risk?<br>

## Key Insights
- Hospitalized patients showed longer recovery periods.<br>
- Severe cases had a higher proportion of long COVID risk.<br>
- Top 10 patients had recovery durations exceeding normal thresholds.<br>
